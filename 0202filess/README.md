# FileManagerApp

Консольное приложение на C#, предназначенное для работы с пользовательскими данными и файлами различных форматов.

Программа позволяет считывать, редактировать, сортировать и сохранять данные пользователей, а также работать с несколькими форматами файлов.

---

# Возможности программы

Приложение реализует следующие функции:

* чтение данных из файла
* запись данных в файл
* вывод данных на экран
* сортировка данных
* поиск данных по подстроке
* добавление новых записей
* удаление записей
* изменение существующих записей

Программа поддерживает работу с несколькими форматами файлов:

* JSON
* XML
* CSV
* YAML

---

# Структура проекта

Проект состоит из трех основных частей.

## FileManagerApp

Основное консольное приложение, которое содержит:

* пользовательское меню
* обработку ввода пользователя
* вызов методов для работы с данными

Файл `Program.cs` реализует:

* главное меню программы
* загрузку и сохранение данных
* поиск, сортировку и вывод пользователей
* добавление, удаление и изменение пользователей

---

## FileManagerLib

Библиотека классов, которая содержит основную логику программы.

В библиотеке реализованы:

### Модель данных

Класс `User`, который содержит информацию о пользователе:

* Id
* Name
* Age
* Role
* RegisterDate
* Activity

Также используется перечисление активности пользователя:

```
Онлайн
Оффлайн
```

---

### Интерфейс работы с файлами

Интерфейс:

```
IFileHandler<T>
```

Он содержит два метода:

```
Read(string filename)
Write(string filename, List<T> data)
```

Этот интерфейс позволяет реализовать универсальную работу с файлами разных форматов.

---

### Обработчики файлов

В библиотеке реализованы классы для работы с различными форматами данных:

#### JsonHandler

Работает с файлами формата JSON.

Использует библиотеку:

```
Newtonsoft.Json
```

Позволяет:

* читать список объектов из JSON
* записывать список объектов в JSON

---

#### XmlHandler

Работает с XML файлами.

Использует стандартный класс:

```
XmlSerializer
```

Позволяет сериализовать и десериализовать список объектов.

---

#### CsvHandler

Реализует чтение и запись CSV файлов.

Особенности:

* данные разделяются символом `,`
* каждая строка соответствует одному пользователю

---

#### YamlHandler

Работает с форматом YAML.

Использует библиотеку:

```
YamlDotNet
```

Позволяет:

* сериализовать данные в YAML
* десериализовать YAML в список объектов

---

# Обработка ошибок

В программе реализована обработка различных исключений:

* проверка существования файла
* ошибки чтения файлов
* ошибки формата данных
* ошибки сериализации

Также добавлено логирование ошибок для дальнейшего анализа.

# Пользовательское меню

После запуска программы пользователю доступно меню:

```
1 Загрузить данные из файла
2 Сохранить данные в файл
3 Показать данные
4 Сортировать по имени
5 Сортировать по возрасту
6 Поиск по имени
7 Добавить пользователя
8 Удалить пользователя
9 Изменить пользователя
0 Выход
```

Каждый пункт вызывает соответствующую функцию программы.

# Unit-тестирование

Для проверки корректности работы программы реализованы Unit-тесты.

Тесты находятся в проекте:

```
FileManagerTest
```

Используется библиотека:

```
MSTest
```

Тестируются следующие компоненты:

### Обработчики файлов

* JsonHandler
* XmlHandler
* CsvHandler
* YamlHandler

Проверяется:

* корректность записи данных
* корректность чтения данных
* соответствие количества объектов

---

### Работа с пользователями

Тестируются методы:

* добавление пользователя
* удаление пользователя
* редактирование пользователя
* сортировка
* поиск

Это позволяет убедиться, что все основные функции программы работают корректно.
